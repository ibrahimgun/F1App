# Formula 1 Schedule App
It is a Formula 1 application that I made to improve my iOS development skills.

## Technical specs
 - Framework: **Alamofire**
 - Dependency Manager: **CocoaPods**
  - API: **[Ergast API](https://ergast.com/mrd//)**
 - Architecture: **MVVM**
